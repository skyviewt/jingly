HearthBuilder - 2014

HearthBuilder is an helper app intended to help players of the popular online card game Hearthstone: Heroes of Warcraft towards the Arena Mode.

It allows users to make guided choice on deck building with help of the AI. The scores of cards are calculated based on mana curve, card statistics and a preselected tier list. Supports URL perssistance that could allow sharing and modifying of the current deck at any point. Registered user can also save their Arena deck and track its stats.

Frontend using Angular.js, backend PHP with CodeIgniter, MySQL with PHPMyAdmin.
